# 山海经宠物桌面精灵 🐕

基于 Electron 开发的 Windows 桌面宠物应用，支持自定义素材上传、互动按钮、随机回应，让你的桌面上多一只可爱的山海经小精灵。

---

## 功能特性

| 功能 | 说明 |
|------|------|
| 🪟 **透明悬浮窗** | 无边框透明窗口，始终置顶，不遮挡工作区 |
| 🎨 **自定义素材** | 支持上传 webp / webm / mp4 / mov / gif / png / jpg / apng |
| 🔀 **状态系统** | 6 种预设状态（正常/玩耍/睡觉/开心/进食/生气），每个状态可绑定不同素材 |
| 😴 **自动睡眠** | 无操作后自动进入睡觉状态，显示 ZZZ 动画和打鼾泡泡 |
| 🖱️ **鼠标拖动** | 拖拽桌宠到屏幕任意位置 |
| 👆 **点击互动** | 单击触发「玩耍」状态 + 随机文字回应 + 弹跳动画 |
| 🔘 **互动按钮** | 悬停显示自定义按钮，每个按钮可绑定不同的触发状态 |
| ⚙️ **控制面板** | 右键或齿轮按钮打开，管理素材/状态映射/所有设置 |
| 💬 **闲置消息** | 正常状态下定时弹出萌宠消息，间隔可自定义 |
| 📏 **大小调节** | 80px ~ 400px 自由缩放，实时生效 |
| 💾 **离线存储** | 所有配置和素材本地存储，修改即时生效无需重启 |

## 快速开始

### 环境要求

- **Node.js** >= 18
- **Windows 10 / 11**（透明窗口仅限 Windows）
- **npm** 或 **yarn**

### 安装

```bash
# 进入项目目录
cd shanhailing-pet

# 安装依赖
npm install

# 启动应用
npm start
```

### 全局快捷键

| 快捷键 | 功能 |
|--------|------|
| `Ctrl + Shift + P` | 打开控制面板 |
| `Ctrl + Shift + S` | (桌宠窗口) 强制进入睡觉状态 |

## 使用指南

### 1. 上传素材

- 右键桌宠 → 选择「控制面板」
- 在「素材管理」区域点击上传区域
- 选择你的图片/视频/GIF 文件
- 支持格式：`.webp` `.webm` `.mp4` `.mov` `.gif` `.png` `.jpg` `.jpeg` `.apng`

> **提示**：使用透明背景的 GIF 或 APNG 可获得最佳悬浮效果。删掉素材会自动解绑所有关联状态。

### 2. 状态素材映射

在「状态素材映射」区域，为每个宠物状态绑定对应素材：

| 状态 | 图标 | 说明 | 触发方式 |
|------|------|------|----------|
| 正常待机 | 🐕 | 默认闲置，轻缓浮动 | 无操作 / 其他状态结束后恢复 |
| 玩耍互动 | 🎾 | 活泼蹦跳 | 单击桌宠 / 「击掌」按钮 |
| 睡觉休息 | 💤 | 闭眼呼吸，显示 ZZZ | 空闲超时自动进入 / `Ctrl+Shift+S` 强制 |
| 开心撒娇 | 💕 | 欢快摇摆 | 「摸摸头」按钮 |
| 进食干饭 | 🍖 | 鼓动进食 | 「喂零食」按钮 |
| 生气炸毛 | 💢 | 抖动炸毛 | 可自定义按钮绑定 |

每个状态可选择「默认占位」或任意已上传素材。状态之间自动切换（如玩耍 3 秒后自动恢复正常）。

### 3. 自定义互动

在控制面板中：

- **互动按钮**：添加/删除/编辑按钮标签,选择触发哪个状态,设置随机回复
- **点击回应**：设置单击桌宠时的随机回复文字（触发「玩耍」状态）
- **闲置消息**：设置正常状态下定时弹出的消息及间隔时间
- **睡眠设置**：调节无操作多久后自动进入睡觉状态（10~120 秒可调）

多条回复用英文逗号 `,` 分隔。

### 4. 宠物状态变化一览

```
正常待机 ──单击/互动按钮──→ 玩耍/开心/进食 ──3~4秒──→ 正常待机
正常待机 ──空闲超时──────→ 睡觉休息 ──任意交互──→ 正常待机
睡觉状态 ──单击/按钮────→ 先唤醒再触发互动
```

### 5. 调节大小

- 在控制面板拖拽「显示大小」滑块
- 范围：80px ~ 400px
- 修改后桌宠即时更新

### 6. 保存设置

- 点击「保存设置」按钮，或按 `Ctrl + S`
- 设置即时生效，桌宠自动刷新

## 技术架构

| 层级 | 技术 |
|------|------|
| **框架** | Electron 28.x |
| **窗口** | BrowserWindow（透明 / 无边框 / 置顶） |
| **通信** | IPC（contextBridge + ipcRenderer/ipcMain） |
| **存储** | 本地文件系统（`userData` 目录） |
| **动画** | CSS3 @keyframes |
| **UI** | 纯 HTML/CSS/JS 无额外框架 |

### 数据存储路径

所有配置和素材存储在 Electron 的 `userData` 目录下：

```
%APPDATA%/shanhailing-pet/
├── config.json           # 配置文件
└── media/                # 素材文件目录
    ├── my-pet.gif
    └── ...
```

## 常见问题

**Q: 透明窗口在某些电脑上显示黑背景？**

A: 确保显卡驱动已更新，并在系统设置中开启硬件加速。部分低配集显可能不完全支持透明窗口。

**Q: 视频素材没有透明背景？**

A: 推荐使用 **WebM (VP9 + Alpha)** 格式或 **GIF/APNG** 来获得透明背景效果。MP4/MOV 不支持 Alpha 通道。

**Q: 如何开机自启？**

A: 可将快捷方式放入 Windows 启动文件夹：
`Win + R` → `shell:startup` → 放入 `npm start` 的快捷方式。

**Q: 桌宠遮挡了其他窗口的操作？**

A: 桌宠窗口默认置顶，如需临时操作被遮挡窗口，可将桌宠拖到屏幕边缘。
